MEDD 8999: Get Your Hands Dirty

A Practical Approach to STEM Education

Course Outline

  • Class 1 & 2: Hardware Electronics with Arduino and Micro:bit

    Objective: Learn the basics of microcontroller programming and build simple hardware projects.

    Subtopics:

    • Introduction to microcontrollers: Arduino vs. Micro:bit
    • Basic electronics and Input/Output basics (LEDs, buttons, sensors)
    • Sensors and actuators, Serial Monitor, and Bluetooth communication
    • Debugging hardware and troubleshooting

    Project: Build a temperature-controlled fan with Arduino OR a step counter using Micro:bit.

  • Class 3 & 4: App Design with MIT App Inventor

    Objective: Design simple mobile applications with functionality and interactivity.

    Subtopics:

    • Introduction to MIT App Inventor and event-driven programming
    • Building user interfaces: Buttons, labels, and layouts
    • Using sensors like GPS and accelerometer
    • Data storage: TinyDB for saving app data

    Project: Create a "an App for checking weather".

  • Class 5 & 6: Software Programming with Scratch

    Objective: Explore visual programming to teach computational thinking.

    Subtopics:

    • Introduction to Scratch: Loops, conditionals, and variables
    • Creating animations and interactive stories
    • Advanced Scratch: Lists, broadcasting, and cloning
    • Optional: Interfacing Scratch to Micro:bit

    Project: Develop an interactive game like a maze or quiz game.

  • Class 7 & 8: 3D Modeling for Printing and VR

    Objective: Use 3D modeling tools to create printable objects and virtual environments.

    Subtopics:

    • Introduction to 3D modeling with Blender
    • Design principles for 3D printing
    • Creating virtual environments with Spatial
    • Applications of 3D printing and VR in STEM education

    Project: Design a virtual classroom in Spatial.

Course Deliverables

  • Projects: Complete four hands-on projects (one for each topic).
  • Reflection: Write reflections on how these tools can be used in classrooms.
  • Final Portfolio: Compile all projects, code, and designs into a final portfolio.
